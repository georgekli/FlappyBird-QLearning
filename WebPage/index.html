Flappy Bird using Q-Learning

This project implements an autonomous agent that plays the well known Flappy Bird game

using Q-Learning. The game core is forked from the FlapPyBird game on github. The results of

the agent can be seen through the video, although it is advised to examine them through

the written formats provided below or the dedicated page.
